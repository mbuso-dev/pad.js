# pad.js
Modern full-stack notepad app

A minimal, browser-based notepad built with HTML, CSS, and Vanilla JavaScript using clean fundamentals. 

## Features
-new note button
-autosave
-save button
-minimal UI
-text formatting
-Time and Date
-fast loading
-local storage
-search functionality

## Tech Stack
-HTML
-CSS
-JavaScript
-Browser Local Storage

## Future Improvements

-Dark / Light mode toggle
-Export notes as .txt
-Markdown support
-Cloud sync (Firebase / Supabase)##
